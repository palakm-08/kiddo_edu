Line graphs in matplotlib

Line graphs are one of the most commonly used types of charts in data visualization. They are great for showing trends or changes over time. In this tutorial, you'll learn how to create a basic line graph using Python's Matplotlib library.

When to Use a Line Graph?

import matplotlib.pyplot as plt

# Sample data

Category= ['A', 'B', 'C', 'D', 'E', 'F']
Values = [10, 20, 15, 25, 10, 30]

#use 'Category' as x-axis and 'Values' as y-axis

plt.plot(Category, Values)

plt.show()

Marker : Marker is used to mark the data points

    Types of marker :
  • 'o' - circle
  • '.' - point
  • '+' - plus
  • '*' - star

Linestyle : linestyle in a plot just means how the line connecting the points looks.

    Types of linestyle :
  • '-' - solid line
  • '--' - dashed line
  • ':' - dotted line
